# Product Backlog Priorizado — Pawify

Este documento contiene las Historias de Usuario (HU) organizadas según su nivel de prioridad para el desarrollo del producto, utilizando la metodología **MoSCoW**.

---

## Prioridad: Alta (Must Have)

_Funcionalidades críticas y esenciales para el lanzamiento del Producto Mínimo Viable (MVP)._

| ID       | Historia de Usuario (HU)         | Rol           | Descripción / Objetivo                                                                      |
| :------- | :------------------------------- | :------------ | :------------------------------------------------------------------------------------------ |
| **HU11** | Registrarse en la aplicación     | Usuario       | Como usuario quiero registrarme en la aplicación para crear mi cuenta personal.             |
| **HU12** | Iniciar sesión (Usuario)         | Usuario       | Como usuario quiero iniciar sesión en la aplicación para acceder a mis datos.               |
| **HU03** | Iniciar sesión (Administrador)   | Administrador | Como administrador quiero iniciar sesión en la aplicación para acceder al panel de gestión. |
| **HU13** | Visualizar catálogo de productos | Usuario       | Como usuario quiero visualizar productos para explorar lo que puedo comprar.                |
| **HU04** | Crear productos                  | Administrador | Como administrador quiero crear productos para ofrecerlos en la tienda.                     |
| **HU17** | Agregar al carrito               | Usuario       | Como usuario quiero agregar productos al carrito para comprarlos después.                   |
| **HU18** | Modificar cantidades del carrito | Usuario       | Como usuario quiero modificar cantidades en el carrito para ajustar mi compra.              |
| **HU07** | Visualizar pedidos globales      | Administrador | Como administrador quiero visualizar pedidos para gestionarlos adecuadamente.               |

---

## Prioridad: Media (Should Have)

_Funcionalidades importantes que aportan gran valor, pero no impiden el lanzamiento inicial si se postergan brevemente._

| ID       | Historia de Usuario (HU)        | Rol           | Descripción / Objetivo                                                                    |
| :------- | :------------------------------ | :------------ | :---------------------------------------------------------------------------------------- |
| **HU14** | Buscar productos por nombre     | Usuario       | Como usuario quiero buscar productos por nombre para encontrar rápido lo que necesito.    |
| **HU16** | Filtrar productos por categoría | Usuario       | Como usuario quiero filtrar productos por categoría para explorar secciones específicas.  |
| **HU05** | Editar productos                | Administrador | Como administrador quiero editar productos para mantenerlos actualizados.                 |
| **HU08** | Cambiar estado de un pedido     | Administrador | Como administrador quiero cambiar el estado de un pedido para avanzar su flujo logístico. |
| **HU19** | Ver estado de pedidos propios   | Usuario       | Como usuario quiero ver el estado de mis pedidos para saber cuándo llegará mi compra.     |
